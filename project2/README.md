# Đồ án II

Một ứng dụng Django đơn giản quản lý nhà hàng, bao gồm các app: `accounts`, `restaurant`, `orders`, `reservations`, `blog`, `dashboard`.

## Tóm tắt

- Django project sử dụng Custom User model.
- Mục tiêu: đăng ký/đăng nhập người dùng, quản lý menu, đặt món, đặt bàn và dashboard quản trị.

## Cấu trúc thư mục 

- `accounts/` — user, forms, views, templates đăng ký/đăng nhập, profile.
- `restaurant/` — menu, category, trang chủ.
- `orders/` — đơn hàng, mục đơn hàng.
- `reservations/` — đặt bàn, bảng.
- `dashboard/` & `blog/` — các module bổ trợ.
